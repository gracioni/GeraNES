# Getting Started

## First Launch

When you open GeraNES, the main menu bar is your starting point:

- `File` for opening and closing ROMs
- `Emulator` for save states, pause, reset, region, and improvements
- `Options` for video, audio, and input configuration
- `Hardware` for FDS and VS System actions when supported by the loaded game

## Quick Start

1. Open `File > Open ROM`.
2. Select a `.nes`, `.fds`, `.nsf`, or supported archive file.
3. If the game starts, confirm your input works.
4. Open `Options > Input` if you need to change keyboard or controller bindings.
5. Use `Emulator > Pause`, `Reset`, or save states as needed.

## Default Core Shortcuts

These shortcuts are visible in the UI and are the fastest way to learn the app:

- `Alt+O`: Open ROM
- `Alt+Q`: Quit
- `Alt+S`: Save State
- `Alt+L`: Load State
- `Alt+P`: Pause
- `Alt+F`: Fullscreen

## Recommended First-Time Checks

- Confirm the correct game region under `Emulator > Region`
- Pick a comfortable scaling mode under `Options > Video > Scale Mode`
- Check your audio device under `Options > Audio > Device`
- Configure your controller in `Options > Input`

## If Something Looks Wrong

Go to [Troubleshooting](troubleshooting.md) if:

- the ROM does not open
- controls do not respond
- audio crackles or uses the wrong device
- the image is stretched or filtered in a way you do not want
