# GeraNES User Guide

GeraNES is a cycle-accurate NES emulator focused on hardware-faithful behavior, but it also includes the features most players expect from a desktop emulator: save states, rewind, shaders, palettes, controller mapping, touch controls, NSF playback, FDS support, and netplay.

This guide is for end users. It explains how to use the emulator's features through the app UI, not how to build the project from source.

## What You Can Do

- Load `.nes`, `.fds`, and `.nsf` files
- Open ROMs from `.zip` archives
- Apply `.ips`, `.ups`, and `.bps` patches when they match the ROM name
- Use save states and rewind
- Configure keyboard and controller input
- Change video filters, shaders, palettes, overscan, scaling, and fullscreen mode
- Tune audio output
- Use Famicom Disk System features
- Join or host netplay sessions
