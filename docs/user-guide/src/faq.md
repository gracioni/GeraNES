# FAQ

## Is This Documentation Meant For Developers?

No. This guide is for players and end users.

## Can I Load ZIP Files Directly?

Yes, GeraNES supports loading supported content from `.zip` archives.

## Does GeraNES Support Save States?

Yes. Use `Emulator > Save State`, `Load State`, and `Slot`.

## Does GeraNES Support Rewind?

Yes, when rewind is enabled in the active configuration.

## Can I Change The NES Colors?

Yes. Use `Options > Video > Palette`.

## Can I Use Shaders?

Yes. Use `Options > Video > Shader`.

## Why Can I Not Pause Or Change The ROM In Netplay?

Because some actions are restricted to the session owner or disabled to keep the session synchronized.

## Why Does My FDS Game Need Extra Setup?

Because Famicom Disk System emulation requires the proper BIOS arrangement.
