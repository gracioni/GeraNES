# Audio

## Where Audio Options Live

Open `Options > Audio`.

## Device Selection

`Options > Audio > Device`

If the wrong output device is active, switch it here first. This is the most common fix when audio comes from the wrong speakers or headset.

## Format Options

Depending on the current backend and platform, GeraNES may expose:

- `Sample Rate`
- `Sample Size`
- `Channels`

These settings let you tune the audio output path to your system.

## When To Change Audio Settings

Try changing audio settings if you notice:

- crackling
- stuttering
- missing sound
- the wrong output device

## Practical Advice

- Start with the default device and default format settings
- Change one setting at a time
- If audio breaks after a change, return to the last stable setup
